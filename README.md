# 🌿 Nature ID Bot — Руководство пользователя

Бот для определения живых организмов по фотографии. Распознаёт насекомых, растения, грибы, птиц, пауков, рыб и многое другое. Использует нейросеть iNaturalist и AI-модель Groq для точного определения вида и подробного рассказа о нём на русском языке.

---

## Быстрый старт

Просто отправь фотографию в чат — бот определит вид и расскажет всё самое важное: как выглядит, чем питается, где обитает, съедобен или ядовит, опасен ли для человека.

---

## Команды пользователя

### /start
Запускает бота и показывает главное меню с кнопками. Вызывай эту команду если кнопки меню пропали или бот перестал отвечать.

### /help
Показывает краткую справку прямо в чате.

### /search [название]
Поиск вида по названию в базе iNaturalist. Работает как с латинскими названиями, так и с русскими.

Примеры:
```
/search Apis mellifera
/search богомол
/search мухомор
/search колорадский жук
```

Бот покажет список подходящих видов с научным названием, рангом, количеством наблюдений в мире и ссылкой на Wikipedia.

### /favorites
Показывает список видов, которые ты добавил в избранное. После каждого успешного определения под результатом появляется кнопка «⭐ В избранное».

### /stats
Твоя личная статистика: сколько запросов использовано сегодня, общее количество запросов, дата регистрации.

## Как работает определение

1. Ты отправляешь фото.
2. Бот уменьшает изображение до оптимального размера.
3. Если на фото есть GPS-метка (EXIF) — она извлекается и учитывается при описании.
4. Фото отправляется в iNaturalist CV — специализированную нейросеть для определения живых организмов, обученную на миллионах наблюдений биологов со всего мира.
5. Результат передаётся в AI-модель Groq, которая пишет подробный рассказ на русском языке.
6. Ты получаешь ответ с названием вида, процентом уверенности и описанием.

### Что означают пометки в ответе

**🌿 Название вида (латынь) — Уверенность iNaturalist: XX%**
Вид определён уверенно. Чем выше процент — тем точнее результат.

**🔄 Возможные альтернативы**
Появляется если уверенность ниже 70% — бот показывает другие варианты определения чтобы ты мог сравнить.

**🔍 Вид не определён с достаточной уверенностью**
iNaturalist ответил, но уверенность слишком низкая. Groq всё равно попытается описать что видит на фото.

**🤖 Распознано нейросетью**
iNaturalist был временно недоступен. Groq работал самостоятельно — результат может быть менее точным.

## Лимиты

- До **20 фото в сутки** от одного пользователя. Счётчик сбрасывается в полночь UTC (3:00 по московскому времени).
- Не чаще **одного фото в 10 секунд** — защита от случайных дублей.

Если лимит исчерпан — бот сообщит об этом и укажет когда счётчик обновится.

---

## Советы для лучшего результата

- **Фокус на организме** — чем чётче видно объект, тем точнее результат.
- **Хорошее освещение** — дневной свет лучше вспышки.
- **Несколько ракурсов** — если с первого фото вид не определился, попробуй другой угол.
- **Реальные фото** — живые фотографии определяются точнее чем иллюстрации из книг или скриншоты.
- **Геолокация** — если в настройках камеры включены GPS-метки, бот учтёт местонахождение при описании вида.

## Часто задаваемые вопросы

**Бот ошибся с определением — что делать?**
Воспользуйся `/search` чтобы уточнить вид вручную. Также попробуй отправить другое фото того же объекта.

**Почему бот говорит «не уверен»?**
Это честная оценка. Если на фото плохо виден объект или вид редкий — iNaturalist возвращает низкий процент уверенности. Бот не будет называть вид наобум.

**Можно ли пользоваться ботом в группе?**
Да. Каждый участник имеет свой независимый лимит и своё избранное.

**Отправил фото, бот не ответил.**
Попробуй через минуту — возможно временная нагрузка на серверы. Если проблема повторяется — напиши администратору.

**Бот не принимает видео, документы, голосовые.**
Бот работает только с фотографиями. Отправь именно фото, не файл.

## Конфиденциальность

Бот сохраняет: Telegram ID, имя пользователя, количество запросов и результаты определений (название вида, уверенность, время ответа). Фотографии **не хранятся** — они обрабатываются и удаляются из памяти сразу после получения ответа.

### Настройки (`/admin/settings`)

Управление параметрами без перезапуска бота:

**score_threshold** — минимальная уверенность iNaturalist (0.0–1.0) для уверенного ответа. По умолчанию 0.40. При значении 0.70 бот будет реже ошибаться, но чаще говорить «не уверен».

**daily_limit_default** — лимит запросов по умолчанию для новых пользователей.

**resize_max_kb** — максимальный размер фото в KB перед отправкой в API. Уменьшение экономит токены Groq.

**groq_model** — модель Groq. Можно сменить без перезапуска.

**bot_active** — глобальный выключатель (true/false). При значении false бот отвечает всем пользователям сообщением о техническом обслуживании.

### Получение OAuth токена iNaturalist (один раз)

Бот использует JWT-аутентификацию iNaturalist. JWT живёт 24 часа и обновляется автоматически. Для его получения нужен долгоживущий OAuth токен — его получают один раз вручную.

**Шаг 1.** Войди на [inaturalist.org](https://www.inaturalist.org).

**Шаг 2.** Создай приложение по адресу `https://www.inaturalist.org/oauth/applications/new`:
- Name: InsectIDBot (любое название)
- Redirect URI: `urn:ietf:wg:oauth:2.0:oob`

Нажми **Create Application** — получишь `client_id` и `client_secret`.

**Шаг 3.** Открой в браузере (подставь свой `client_id`):
```
https://www.inaturalist.org/oauth/authorize?client_id=ВАШ_CLIENT_ID&redirect_uri=urn:ietf:wg:oauth:2.0:oob&response_type=code
```
Нажми «Authorize» — получишь **authorization code**.

**Шаг 4.** В терминале:
```bash
curl -X POST https://www.inaturalist.org/oauth/token \
  -d "client_id=ВАШ_CLIENT_ID" \
  -d "client_secret=ВАШ_CLIENT_SECRET" \
  -d "code=ВАШ_AUTHORIZATION_CODE" \
  -d "redirect_uri=urn:ietf:wg:oauth:2.0:oob" \
  -d "grant_type=authorization_code"
```

В ответе получишь JSON с полем `access_token` — это и есть OAuth токен.

**Шаг 5.** Вставь токен в `INAT_API_KEY` на Render.com. Дальше бот сам обновляет JWT каждые 23 часа.
